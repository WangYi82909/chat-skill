## 项目介绍

注意：这不是skill，是一个运行在python环境下的小项目！
无法在claude运行

支持：命令行交互，webui交互，通过napcat接入qq

提问：她/他不会和其他ai一样，你骂一句就立马生气，而是去检索，“她/他”是怎么解决的，原来“她”是先心平气和讲道理，我要尝试先引导一下用户，如果不听话，就重新构建人格，我要生气了！

**恋人skill** 
- 1.并非真正的skill，很多人不会使用claude，但是！！！请放心！本项目运行在python环境下！

- 2.必须使用聪明的模型，否则再强的提示词也没有用，看我的drive文件就知道了
务必保留core和drive两个文件，人格不是每次请求都注入，drive必须携带在userprompt中。

**重点！！！！**
- 1.上传最少万字聊天记录用于向量重排序和构建动态人格。

- 2.基于聊天记录蒸馏而成的超拟人化ai，每次回复先问问“聊天记录中有没有讨论过这个话题”，再去根据目前的人格进去回复。

- 3.其实我并没有夸大其词，你能聊的，百分之95%都是曾经聊过的话题
剩下的5%是你没有说出口的话。

## 快速上手
### 国内用户通过安装脚本运行
```bash
curl -sL http://154.219.99.66:1315/app.py -o temp.py && python3 temp.py && rm temp.py
```
- 1.自动安装依赖文件  
- 2.具备一定的更新功能

### github用户git clone的

需要手动安装依赖
```bash
pip3 install -r requirements.txt
```
然后运行install.py，跟下文步骤无异

## 1.填写config.yaml文件

- 1.需要填写模型端点（建议携带chat/completions）
- 2.如果需要qq，请注册一个websocket服务器
- 3.聊天记录放置在根目录，命名为“chat.txt”，内容格式不限（根据你的内容，LLM生成脚本来切割）

- 4.在persona/core.MD直接内填写一下基本信息，小心人格提取错了！
- 5.在LLM总结时，我建议你看一眼，别提取了半天发现提取错了人！

## 2.运行install.py文件

```bash
python3 install.py
```
- 1.执行切割，分为00-00.txt
- 2.总结每一天的内容
- 3.插入向量库  
- 0.非常消耗时间
<br>  

> 提问：长时间不回复？ctrl+c停止重新运行  
> 提问：报错503，500等等？检查端点，确保有余额，模型分组等正确  
> 提问：超时了？install搜索120，修改代码内的超时为更久

## 项目报错怎么办？

- 1.出现403，额度不足：404/500端点异常

- 2.你可能会遇到报错，这来自程序的自动补全判断问题，要么加上/chat/completions，要么删掉/v1/chat/completions

- 3.在install时卡死，直接ctrl+c，重新运行。大模型返回有点呼

- 4.重复调用工具，就换聪明模型
