## 项目介绍

**超越前任skill** 是一个专注于**基于真实聊天记录蒸馏人物灵魂**的 AI 终端交互引擎。

这不仅仅是一个简单的 LLM 对话脚本，而是一个具备“长期记忆检索”与“动态人格生长”能力的赛博容器。系统会读取你前期通过处理海量聊天记录清洗、总结并结构化提取出的“技能（skill）”与特征文件，将其作为核心驱动力。

在每一轮的交互中，AI 会根据当前的语境，自动思考并调用工具去检索过往的聊天记录（记忆）。如果找到了熟悉的话题，它会用最贴合当时的语气回应你；如果没有，它会基于蒸馏出的底层人格特征，替你说出那些曾经未曾开口的 5%。

### 核心机制：人格蒸馏与动态注入

* **实时文件读取 (Real-time Context):** 系统在每一轮推理前，都会实时从磁盘读取 `core.MD`（核心人格）和 `drive.MD`（驱动指令）。这意味着 AI 的性格可以在对话进行中被动态热更新，无需重启服务。
* **工具调用循环 (Tool Calling Loop):** 内置最大 6 轮的思考循环。AI 会在回复你之前，自行决定是否需要调用 `query` 工具去翻阅历史聊天库，或者调用 `search` 获取外部信息。
* **人格修正 (Dynamic Correction):** 独特的 `correction` 工具允许 AI 根据对话走向，动态修改自身的情绪和行为标签，并实时回写到人格配置中，实现真正的“人格演进”。
* **记忆缓存屏障 (Cache Barrier):** 每一轮的工具调用结果都会被临时缓存，防止 AI 陷入死循环重复检索，保证对话的流畅性与算力的有效利用。

## 功能特性

* **记忆外挂机制:** 自动识别当前意图，提取关键词去向量库或数据库中执行 `query`。
* **结构化特质驱动:** 完美兼容从海量历史对话中提取出的结构化人物性格文件。
* **思考可见性:** 控制台全量打印 AI 的思考过程、工具调用理由（`why?`）以及内部缓存状态，让你看清它的每一次“回忆”。
* **沉浸式终端 UI:** 预设丰富的 ANSI 控制台颜色，INFO、TOOL、WARN、USER 分级输出，界面清爽且极具极客感。
* **对话日志固化:** 每一轮的 Prompt 上下文与 LLM 原始回复都会带有时间戳自动落盘到 `logs/` 目录下，方便复盘与二次数据蒸馏。
